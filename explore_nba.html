NBA Scorigami Analysis

A Scorigami is a concept (popularized by sports writer Jon Bois) describing a final scoreline that has never occurred before in a league’s history. Because scores cluster heavily around the league average (currently ~110-115 points per team), scorelines in this range are common, while extremely lopsided, low-scoring, or high-scoring outcomes are rare and establish new Scorigami points.

Here, we calculate and plot the historical record of unique scorelines in the NBA from 1946 to 2026.

Code 
# Setup scorigami data
min_s <- 15
max_s <- 190

# 1. Home vs Visitor Score combinations
home_visitor_scores <- games_clean %>%
  filter(homeScore >= min_s & homeScore <= max_s & awayScore >= min_s & awayScore <= max_s) %>%
  group_by(homeScore, awayScore) %>%
  summarise(
    Count = n(),
    First_Date = min(gameDate),
    Last_Date = max(gameDate),
    .groups = "drop"
  )

# 2. Winner vs Loser Score combinations (w >= l)
winner_loser_scores <- games_clean %>%
  filter(homeScore >= min_s & homeScore <= max_s & awayScore >= min_s & awayScore <= max_s) %>%
  mutate(
    WinnerScore = pmax(homeScore, awayScore),
    LoserScore = pmin(homeScore, awayScore)
  ) %>%
  group_by(WinnerScore, LoserScore) %>%
  summarise(
    Count = n(),
    First_Date = min(gameDate),
    Last_Date = max(gameDate),
    .groups = "drop"
  )
